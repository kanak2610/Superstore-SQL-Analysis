# Superstore SQL Analysis

## Objective
Analyze sales data using SQL using filtering, aggregation, sorting, and business queries.

## Dataset
Superstore Dataset (Kaggle)

## Tools Used
- SQLite
- DB Browser for SQLite
- GitHub

## Tasks Performed
- Loaded dataset into SQLite database
- Explored schema and sample data
- Applied WHERE filters
- Used GROUP BY for aggregations
- Identified top customers and products
- Performed duplicate checks
- Validated data quality

## Key Insights
- Dataset contains 9995 records.
- Sales were analyzed across regions and categories.
- Top customers and products were identified.
- Data quality checks were performed.
