#SQL Sales Analysis Project

## Objective
Analyze sales data using SQL with filtering, aggregation, sorting, and business queries.

## Dataset
Superstore Dataset

## Tasks Performed
- Loaded dataset into SQL database
- Explored schema and sample records
- Applied WHERE filters
- Performed GROUP BY aggregations
- Identified top products and categories
- Analyzed monthly sales trends
- Found top customers
- Checked duplicates and data quality

## Files Included
- SampleSuperstore.csv
- sales_analysis.sql
- Screenshots of outputs
- insights.md

## Tools Used
- SQL
- MySQL / SQLite
- GitHub

## Key Insights
- Technology category generated high sales.
- Top customers contributed major revenue.
- Monthly sales trends revealed peak periods.
- Dataset quality validated successfully.
